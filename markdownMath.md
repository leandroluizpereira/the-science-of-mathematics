
Para deixar o número  subscrito use a tag html ``` 2 <sub> x </sub>  ```

Ficando : 2 <sub> x </sub>

Para deixar  númro sobrescrito ``` 2 <sup> x</sup> ```

Ficando : 2 <sup> x </sup>

∉	não pertence
∈	pertence
∞	infinito
↔	seta dupla

⊂	contido
∪	união
Σ	somatória (sigma)
